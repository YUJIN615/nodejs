hhtps와 http2

http
- 웹 서버에 SSL 암호화를 추가하는 모듈
- 오고 가는 데이터를 암호화해서 중간에 다른 사람이 요청을 가로채더라도 내용을 확이할 수 없음
- 요즘에는 https 적용이 필수(개인 정보가 있는 곳은 특히)
- 쿠키, 쿼리스트링 등에 로그인 비밀번호가 들어있어서 탈취가 될 수 있음
- 이 때 https를 사용하면 암호화 되어있어서 안전함

http2
- SSL 암호화와 더불어 최신 HTTP 프로토콜만 http/2를 사용하는 모듈
- 요청 및 응답 방식이 기존 http/1.1 보다 개선됨
- 웹의 속도도 개선됨
- 동시에 여러개 요청보낼 수 있게 됨
- 실무에서 적용할 수 있으면 적용하는게 좋다.
- 속도와 보안 둘 다 개선할 수 있다.
- 개발할 때 http로 하고 배포할 때 http2로
